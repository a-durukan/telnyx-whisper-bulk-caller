Welcome to Telnyx Whisper Bulk Caller

This application allows you to make bulk phone calls, record them, and transcribe the recordings using AI.

Features

  • Bulk Calling: Make multiple calls simultaneously with Telnyx API
  • Auto Recording: Automatically record all calls
  • AI Transcription: Transcribe recordings using OpenAI Whisper
  • Audio Transcriber: Batch transcribe audio files
  • Results Export: Download results as TSV files

Getting Started

  1. Go to Configuration and set up your API keys
  2. Use Bulk Caller to start making calls
  3. Monitor and download results from Call Results
  4. Use Audio Transcriber for batch audio transcription
